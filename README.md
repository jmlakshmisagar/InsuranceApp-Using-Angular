## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/` | Get all employees |
| POST | `/create` | Create new employee |
| GET | `/find/{id}` | Find employee by ID |
| PUT | `/update` | Update employee |
| DELETE | `/delete/{id}` | Delete employee |
